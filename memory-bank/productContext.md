# Product Context

## Проблема которую решает проект
Трейдеры не могут быстро проверить торговые идеи без глубокого знания кода.
Результаты бэктестов расходятся с TradingView — нет доверия к метрикам.
Ручной подбор параметров занимает часы.

## Как решаем
1. **Strategy Builder** — drag-and-drop блоки (RSI, MACD, Supertrend и 40+ индикаторов)
   → соединяешь блоки → жмёшь Run → получаешь метрики
2. **Parity Engine** (FallbackEngineV4) — точное воспроизведение логики TradingView
3. **Optimizer** — Optuna автоматически ищет лучшие параметры (Sharpe, CalmarRatio, etc.)
4. **AI Agents** — DeepSeek/Qwen генерируют стратегии из текстового описания

## Пользователи
- Трейдеры без опыта кодинга (Strategy Builder)
- Python-разработчики (прямой API)
- Исследователи (оптимизация, Monte Carlo, Walk-Forward)

## Ограничения
- Только Bybit (spot/linear perpetuals)
- Данные с 2025-01-01 (DATA_START_DATE)
- Максимум 730 дней бэктеста
- Только таймфреймы: 1, 5, 15, 30, 60, 240, D, W, M
