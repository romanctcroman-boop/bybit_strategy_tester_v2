# Планы задач (.cursor/plans/)

Здесь сохраняются планы сложных задач перед выполнением.

- **Формат:** один файл на задачу, имя в латинице без пробелов (например: `refactor-backtest-engine.md`, `add-rsi-strategy-params.md`).
- **Содержимое:** анализ (файлы, переменные, зависимости), декомпозиция (подзадачи), порядок выполнения, точки валидации.
- **Правила:** см. `.cursor/rules/planning.mdc`, `task-decomposition.mdc`, `task-breakdown.mdc`.
- **Шаблон:** см. `.cursor/commands/analyze-task.md`.

После выполнения план можно оставить для истории или удалить.
