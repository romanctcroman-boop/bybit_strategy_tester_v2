## 🚀 Usage Examples

### Basic Benchmark:

```python
from backend.benchmarking.performance import BenchmarkSuite

suite = BenchmarkSuite()

# Sync benchmark
def slow_function():
    time.sleep(0.01)

result = await suite.run_benchmark(
    "slow_function",
    slow_function,
    iterations=100,
)

print(f"Avg: {result.avg_time*1000:.2f}ms")
print(f"P95: {result.p95_time*1000:.2f}ms")
```

### Async Benchmark:

```python
async def api_call():
    await asyncio.sleep(0.01)

result = await suite.run_benchmark(
    "api_call",
    api_call,
    iterations=100,
)
```

### Load Testing:

```python
result = await suite.run_load_test(
    api_call,
    concurrent_users=10,
    duration_seconds=60,
)

print(f"RPS: {result.requests_per_second:.1f}")
print(f"Error rate: {result.error_rate:.0%}")
```

### Regression Detection:

```python
# Set baseline
suite.set_baseline("api_call", baseline_result)

# Check for regression
is_regression, message = suite.check_regression(
    "api_call",
    current_result,
    threshold_percent=0.2  # 20%
)

if is_regression:
    print(f"⚠️ {message}")
```

### Bottleneck Detection:

```python
bottlenecks = suite.detect_bottlenecks()

for bottleneck in bottlenecks:
    print(f"⚠️ {bottleneck}")
```

### Generate Report:

```python
report = suite.generate_report()

print(f"Passed: {report.passed_benchmarks}/{report.total_benchmarks}")
print(f"Bottlenecks: {len(report.bottlenecks)}")
print(f"Recommendations: {len(report.recommendations)}")
```

## 📈 Metrics

### Benchmark Statistics:

- **avg_time** — Average response time
- **min_time** — Minimum response time
- **max_time** — Maximum response time
- **median_time** — Median response time
- **std_dev** — Standard deviation
- **p95_time** — 95th percentile
- **p99_time** — 99th percentile
- **success_rate** — Success rate (0.0-1.0)

### Load Test Statistics:

- **concurrent_users** — Number of concurrent users
- **total_requests** — Total requests
- **successful_requests** — Successful requests
- **failed_requests** — Failed requests
- **requests_per_second** — Throughput

- **p95_response_time** — 95th percentile response time
- **p99_response_time** — 99th percentile response time
- **error_rate** — Error rate (0.0-1.0)

## 🎯 Key Features

### 1. Response Time Benchmarks:

```python
result = await suite.run_benchmark(
    "function_name",
    function_to_test,
    iterations=100,
    warmup_iterations=10,
)
```

### 2. Load Testing:

```python
result = await suite.run_load_test(
    function_to_test,
    concurrent_users=50,
    duration_seconds=120,
)
```

### 3. Regression Detection:

```python
# Set baseline after good performance
suite.set_baseline("critical_function", good_result)

# Check for regression in CI/CD
current_result = await suite.run_benchmark(...)
is_regression, msg = suite.check_regression("critical_function", current_result)

if is_regression:
    exit(1)  # Fail CI/CD
```

### 4. Bottleneck Detection:

Automatically detects:
- High variance (inconsistent performance)
- High tail latency (p99 > 3x avg)
- Low success rate (< 95%)
- High error rate under load (> 5%)
- Response time degradation under load

### 5. Comprehensive Reporting:

```python
report = suite.generate_report()

# Access all data
print(report.results)  # Benchmark results
print(report.load_tests)  # Load test results
print(report.bottlenecks)  # Detected bottlenecks
print(report.recommendations)  # Optimization recommendations
```
