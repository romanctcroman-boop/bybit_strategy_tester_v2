### P2-8: L2 order book ✅

**Созданные файлы:**
- `backend/experimental/l2_lob/collector.py` — WebSocket collector

**Функционал:**
- WebSocket подключение к Bybit
- Real-time order book data
- Snapshot collection
- Metrics calculation (spread, imbalance)
- Save/load snapshots

**Пример использования:**
```python
from backend.experimental.l2_lob.collector import L2OrderBookCollector

collector = L2OrderBookCollector(symbol='BTCUSDT', testnet=True)
await collector.start()

# Get current snapshot
snapshot = collector.get_current_snapshot()

# Get metrics
metrics = collector.calculate_metrics()
print(f"Spread: {metrics['spread_pct']:.2f}%")
print(f"Imbalance: {metrics['imbalance']:.2f}")
```
