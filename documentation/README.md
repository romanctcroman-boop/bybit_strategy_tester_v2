# Documentation Catalog

This directory contains the technical documentation for **Bybit Strategy Tester v2**.

- `technical_specifications.md` – detailed functional and non‑functional requirements for each UI page.
- `ui_sketches.md` – visual mockups (PNG images) referenced from the design repository.
- `implementation_plan.md` – phased rollout plan with milestones and dependencies.

All files are written in Markdown and follow the project’s documentation standards.
